## 2. 시험 대상 시스템

ePub 2.0 이하 전자책을 ePub 3.0 인터랙티브 콘텐츠로 자동 변환하는 웹 기반 AI 시스템입니다.

- **기술 스택**: TypeScript, Node.js 20, Next.js 16, Google Gemini AI
- **배포 환경**: Vercel (https://gov-epub-2026-three.vercel.app)
- **소스코드**: GitHub 비공개 저장소 (접근 권한 제공 가능)

## 3. 시험 요청 항목 (KPI 13개)

| # | KPI 항목 | 목표치 | 측정 방법 |
|---|----------|--------|-----------|
| 1 | ePubCheck 통과율 | >= 95% | 샘플 ePub 자동 변환 후 ePubCheck 실행 |
| 2 | 퀴즈 HTML 구조/문법 오류율 | <= 1% | W3C Validator 검증 |
| 3 | 퀴즈 JSON 스키마 검증 통과율 | >= 98% | JSON Schema Validator |
| 4 | TTS 텍스트 싱크 정확도 | >= 98% | 타임스탬프 오차 측정 |
| 5 | TTS 무음 구간 비율 | <= 5% | 오디오 분석 |
| 6 | KWCAG 접근성 충족율 | >= 90% | KWCAG 2.1 자동 검증 + Ace by DAISY |
| 7 | 인터랙션 요소 자동 포함 수 | >= 3종/권 | 변환 로그 분석 |
| 8 | API 평균 응답시간 | <= 3초 | 서버 로그 측정 |
| 9 | 시스템 가용률 | >= 99.5% | 업타임 모니터링 |
| 10 | 자동 테스트 케이스 통과율 | >= 90% | Vitest 60개 TC 실행 |
| 11 | 변환 후 구조 오류 발생률 | <= 2% | ePubCheck + 커스텀 검증 |
| 12 | GitHub Actions 테스트 자동 통과율 | 100% | CI/CD 파이프라인 실행 |
| 13 | 문서화 커버리지 | >= 3건 | 문서 건수 확인 |
