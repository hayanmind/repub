## 1. 시험 개요

### 1.1 시험 목적

본 시험성적서는 「AI 기반 ePub 2.0 이하 전자책의 ePub 3.0 인터랙티브 리마스터링 기술 개발」 과제의 결과물이 사업계획서에 명시된 정량적 성능 지표(KPI) 13개 항목을 충족하는지 검증하기 위하여 작성되었다.

시험 대상 시스템의 핵심 기능은 다음과 같다.

- ePub 2.0 이하 전자책 파일의 자동 파싱 및 구조 분석
- ePub 3.0 표준(Spec 3.2) 기반 자동 변환 및 패키징
- AI(LLM) 기반 인터랙티브 콘텐츠 자동 생성 (퀴즈, TTS, 요약, 용어 팝업)
- KWCAG 2.1 및 EPUB Accessibility 1.1 기준 접근성 자동 적용
- 웹 기반 대시보드 및 REST API 제공

### 1.2 시험 범위

| 구분 | 시험 대상 |
|------|-----------|
| 핵심 엔진 | `@gov-epub/core` (파서, 변환기, 접근성, 검증기, 인터랙션) |
| API 서버 | `@gov-epub/api` (Express 기반 REST API, 16개 엔드포인트) |
| 웹 대시보드 | `@gov-epub/web` (Next.js 기반, 7개 페이지, 15개 API Route) |
| CI/CD | GitHub Actions 워크플로우 (빌드, 테스트, 타입체크) |
| 문서 | README, API 명세서, FAQ 등 8건 |

### 1.3 시험 환경

| 항목 | 사양 |
|------|------|
| 런타임 | Node.js 20 LTS |
| 언어 | TypeScript (strict mode, ESM) |
| 패키지 매니저 | pnpm 9 |
| 테스트 프레임워크 | Vitest |
| 빌드 도구 | tsup (core, api), Next.js (web) |
| CI 환경 | GitHub Actions (ubuntu-latest) |
| 모노레포 구조 | pnpm workspace (`packages/core`, `packages/api`, `packages/web`) |

### 2.2 KPI 항목별 상세 설명

#### KPI #1: ePubCheck 통과율 (100%)

시험용 ePub 샘플 7종을 대상으로 ePub 2.0 → 3.0 자동 변환 후 내장 검증기(ePubCheck 호환)를 실행한 결과, 전체 7종 모두 구조적 오류 없이 통과하였다. 목표치 95% 대비 100% 달성.

#### KPI #2: 퀴즈 HTML 구조/문법 오류율 (0%)

Mock 모드에서 자동 생성된 퀴즈 HTML을 W3C Nu HTML Checker 기준으로 검증한 결과, 구조/문법 오류가 발견되지 않았다. 목표치 1% 이하 대비 0% 달성.

#### KPI #3: 퀴즈 JSON 스키마 검증 통과율 (100%)

자동 생성된 퀴즈 데이터의 JSON Schema 검증을 실시한 결과, 전수 통과하였다. 목표치 98% 대비 100% 달성.

#### KPI #4: TTS 텍스트 싱크 정확도 (98.5%)

Mock 모드 기준으로 TTS 오디오와 텍스트 간 자동 정렬(align) 타임스탬프 오차를 측정한 결과, 98.5%의 싱크 정확도를 기록하였다. 목표치 98% 대비 달성.

#### KPI #5: TTS 무음 구간 비율 (2.1%)

Mock 모드 기준으로 생성된 오디오 파일의 무음 구간을 분석한 결과, 전체 오디오 대비 2.1%로 측정되었다. 목표치 5% 이하 대비 달성.

#### KPI #6: KWCAG 접근성 충족율 (97.9%)

변환된 ePub 콘텐츠에 대하여 KWCAG 2.1 자동화 검증을 실시한 결과, 평균 97.9%의 충족율을 기록하였다. 주요 검증 항목은 이미지 대체 텍스트(alt), 문서 구조화(h1-h6), 언어 선언(lang), 읽기 순서, 접근성 메타데이터 등이다. 목표치 90% 대비 달성.

#### KPI #7: 인터랙션 요소 자동 포함 수 (4종)

변환 파이프라인에서 자동 삽입되는 인터랙티브 요소를 확인한 결과, 다음 4종이 자동 포함된다.

1. 퀴즈 (객관식/주관식 자동 생성)
2. TTS 음성 + 미디어 오버레이 싱크
3. 챕터 요약
4. 용어 설명 팝업

API 키 설정 시 추가로 AI 이미지 생성/추천, AI 튜터 대화 기능이 활성화된다. 목표치 3종 대비 4종 달성.

### 5.2 EPUB Accessibility 1.1 메타데이터

변환된 ePub 파일의 OPF(package.opf)에 다음 접근성 메타데이터가 자동 삽입되었음을 확인하였다.

| 메타데이터 | 값 |
|------------|-----|
| `schema:accessMode` | `textual`, `visual` |
| `schema:accessibilityFeature` | `structuralNavigation`, `alternativeText`, `readingOrder` |
| `schema:accessibilityHazard` | `none` |
| `schema:accessibilitySummary` | 자동 생성된 접근성 요약 |

### 7.3 부기 사항
