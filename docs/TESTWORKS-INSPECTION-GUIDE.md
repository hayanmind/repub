## 3. 일반 진행 타임라인

공인 시험성적서 발급은 평균 **3~6주**가 소요됩니다. 2026.4.30 기한을 준수하려면 **2026.3.초**에는 견적 의뢰를 착수해야 합니다.

| 단계 | 소요 기간 | 주요 활동 |
|------|-----------|-----------|
| **① 견적 의뢰** | 1~3 영업일 | 견적 이메일 발송, 구두 상담, 대략적 비용/일정 확인 |
| **② 견적서 수령** | 3~5 영업일 | 공식 견적서 수령 (계약 조건 포함) |
| **③ 계약 체결** | 3~5 영업일 | 시험 의뢰서/계약서 작성, 기관 내부 검토, 날인 |
| **④ 시험 착수 회의** | 1일 | 시험 범위/방법 확정, 검수인 지정, 현장 방문 일정 확정 |
| **⑤ 시험 수행** | 3~10 영업일 | 검수인이 시험 수행 (현장 또는 원격). 결과물 검증 |
| **⑥ 시험 보고서 초안** | 3~5 영업일 | 기관 내부 QA 후 초안 회람 |
| **⑦ 성적서 발급** | 3~5 영업일 | 공식 성적서(서명·날인본) 발급 및 수령 |

**합계**: 대략 **3~6주 (영업일 기준 15~30일)**. 긴급 일정은 가산금(약 20~30%)을 지불하면 2주로 단축 가능한 기관도 있습니다.

## 2. 시험 대상 시스템

ePub 2.0 이하 전자책을 ePub 3.0 인터랙티브 콘텐츠(퀴즈·TTS·접근성 태그 포함)로 자동 변환하는 웹 기반 AI 시스템입니다.

- 기술 스택: TypeScript, Node.js 20 LTS, Next.js 16, Google Gemini / OpenAI / ElevenLabs API
- 배포 URL: https://gov-epub-2026-three.vercel.app (Vercel 서버리스)
- 소스 저장소: GitHub 비공개 (NDA 체결 후 Read-only 권한 제공 가능)

## 3. 시험 요청 범위 (정량 13개 KPI)

| # | KPI | 목표치 | 측정 방법 |
|---|-----|--------|-----------|
| 1 | ePubCheck 통과율 | ≥ 95% | 샘플 100권 변환 후 ePubCheck 4.x 실행 |
| 2 | 퀴즈 HTML 구조/문법 오류율 | ≤ 1% | 변환 결과물 내 퀴즈 블록 well-formedness 검사 |
| 3 | 퀴즈 JSON 스키마 검증 통과율 | ≥ 98% | JSON Schema Validator |
| 4 | TTS 텍스트 싱크 정확도 | ≥ 98% | SMIL par/audio 페어링 + 타임스탬프 단조성 |
| 5 | TTS 무음 구간 비율 | ≤ 5% | SMIL 발화/무음 시간 비율 |
| 6 | KWCAG 접근성 충족율 | ≥ 90% | KWCAG 2.1 자동 + Ace by DAISY |
| 7 | 인터랙션 요소 자동 포함 수 | ≥ 3종/권 | 변환 로그 |
| 8 | API 평균 응답시간 | ≤ 3초 | 서버 로그 기반 자동 측정 |
| 9 | 시스템 가용률 | ≥ 99.5% | 업타임 모니터 |
| 10 | 자동 테스트 통과율 | ≥ 90% | Vitest 60개 TC 실행 |
| 11 | 변환 후 구조 오류율 | ≤ 2% | ePubCheck + 커스텀 검증 |
| 12 | GitHub Actions CI 통과율 | 100% | CI/CD 파이프라인 실행 |
| 13 | 문서화 커버리지 | ≥ 3건 | 문서 건수 확인 |

## 4. 시험 방법

당사에서 검수인이 현장에 도착하신 뒤 **단일 스크립트 실행만으로** 전 KPI 측정이 완료되도록 준비해 두었습니다.

```bash
bash scripts/inspect-run.sh
```

- 소요: 약 20~40분 (대량 데이터셋 제외 기본 시험)
- 산출: `inspection-results/{timestamp}/` 폴더 + ZIP 파일
- 내용: KPI JSON 리포트, Vitest 로그, ePubCheck 리포트, 환경 정보

세부 절차는 `docs/INSPECTOR-ENVIRONMENT.md` 및 `docs/INSPECTOR-CHECKLIST.md` 에 정리되어 있으며, 견적 확정 후 즉시 공유드리겠습니다.

## 5. 예상 비용 범위 (참고치)

> **경고**: 아래는 업계 통상 범위를 기반으로 한 **추정치**이며, 확정 비용은 기관 견적서로 확인해야 합니다.

### 5.1 KPI 13개 기반 기본 시험

| 항목 | 범위 |
|------|------|
| **기본 시험 비용** (KPI 13개, 샘플 7~100권) | **800만 원 ~ 1,500만 원** |
| **대량 시험 옵션** (1,000권 전수) | +300만 원 ~ +500만 원 |
| **접근성 단독 심화 시험** (Ace by DAISY + KWCAG 2.1) | +200만 원 ~ +400만 원 |
| **긴급 일정 가산** (2주 내 발급) | 기본가의 +20~30% |
| **출장비/수행비 가산** (현장 시험 시) | +30~80만 원 |

### 5.2 기관별 참고 단가 수준

| 기관 | 기본 시험 추정가 | 성격 |
|------|-----------------|------|
| TTA | 1,200만 ~ 1,800만 원 | 공신력 최상, 가격 최상위 |
| KTL | 900만 ~ 1,400만 원 | 공인기관, 중상위 |
| 테스트웍스 | 600만 ~ 1,200만 원 | 민간, 접근성 특화 |
| ETRI | 1,500만 ~ 2,500만 원 | 연구기관, 고난도 시험 |

### 5.3 예산 관리 팁

- **정부과제 기성금 승인 일정**을 고려해 계약 체결 시점과 예산 지출 시점을 맞추십시오.
- 세금계산서는 **계약 → 선금 30% + 잔금 70% (성적서 발급 후)** 구조가 일반적입니다.
- 시험비는 **연구개발비 - 외부용역비** 또는 **연구개발비 - 연구활동비** 항목으로 계상 가능합니다 (전문기관 회계 가이드 사전 확인 필수).

## 7. 후속 관리

1. **원본 보관**: 하얀마인드 본사 + 클라우드 이중 백업
