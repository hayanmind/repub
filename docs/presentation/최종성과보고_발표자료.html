01. 과제 개요
구형 ePub 전자책을 AI로 자동 리마스터링
개발 배경
2010년대 제작된 ePub 2.0 전자책은 텍스트 중심의 단순 구조로, 멀티미디어·상호작용·접근성이 부족.
ePub 3.0은 HTML5/CSS3/JS 기반으로 인터랙티브 콘텐츠를 지원하나, 수작업 변환에 1권당 2.5만원 소요.
개발 목표
AI 기반 자동 변환 파이프라인을 구축하여 변환 비용을 80% 이상 절감하고,
퀴즈·TTS·요약·접근성 태깅을 자동 적용하는 원스톱 시스템 개발.

핵심 목표 6가지

1. 구조 자동 변환
ePubCheck 통과율 ≥ 95%
2. LLM 콘텐츠 보강
요약, 퀴즈, 이미지 생성
3. TTS 음성 변환
SMIL 미디어 오버레이 싱크
4. 접근성 자동 적용
KWCAG 2.1 충족률 ≥ 90%
5. 웹 대시보드
SaaS/API 배포
6. 오픈소스 공개
GitHub + 2년 무상 제공

06. Sigil 플러그인 및 CLI
오픈소스 ePub 에디터 Sigil(7,200+ stars) 생태계 연동

Sigil 플러그인

Sigil에서 직접 AI 리마스터링을 실행할 수 있는 output 플러그인.

API 모드: 원격 서버 호출 (Node.js 불필요)
Local 모드: CLI 로컬 실행 (오프라인)
GUI: Tkinter 설정 다이얼로그

Sigil > Plugins > Output > ePubRemaster

기존 플러그인 대비 차별점

기능ePub3-itizerePubRemaster
구조 변환OO

TTS+SMIL-O
접근성 태깅-O
GUI 설정-O

CLI 인터페이스

$ epub-remaster convert input.epub \
    -o output.epub --quiz --summary
